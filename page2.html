CYP.0R1 소개

CYP.0R1이란?

저희 개발 팀은 약물유전체 기반 환자의 유전형별 맞춤형 치료를 위해 프로젝트 CYP.0R1 개발을 진행했습니다.

CYP2C19과 CYP2D6는 약물 대사의 핵심을 담당하는 효소군으로, 사람마다 같은 약이 다르게 작용하는 이유가 이 유전자에 있습니다. 본 프로젝트의 핵심 타겟인 이 효소군인 CYP450에서 'CYP'를 따왔습니다.

또한, 유전자를 다루는 도구에서 영감을 받아 팀명 '오리조'를 유전자 명명법으로 재해석하여 '0R1'를 만들었습니다. 이 둘을 합쳐 CYP.0R1 — 환자의 유전형을 읽고 그 사람에게 맞는 치료를 찾는 정밀함을 이름에 담았습니다.

// 02

약물유전체 기초 정의

Pharmacogenetics

Pharmacogenetics(약물유전체학)는 개인의 유전적 특성이 약물의 대사, 효능, 부작용에 어떤 영향을 미치는지를 연구하는 학문입니다.

같은 약을 같은 용량으로 복용해도 사람마다 효과가 다른 이유는 유전자에 있습니다. 특히 CYP450(Cytochrome P450) 효소군은 체내에서 약물을 대사하는 핵심 효소로, 이 효소를 암호화하는 유전자의 변이에 따라 약물 대사 속도가 크게 달라집니다. 본 프로젝트에서는 CYP2D6, CYP2C19와 같은 주요 CYP450 유전자를 중심으로 데이터베이스를 구축하였습니다.

// 03

핵심 개념

유전체 데이터를 분석하고 알고리즘을 설계하기 위해 명확한 유전학적 단위 표준화가 필요합니다.

🧬

대립유전자 (Allele)

하나의 유전자는 위치(locus)에 따라 다양한 형태로 존재할 수 있으며, 이 각각의 형태를 allele이라고 합니다. 예를 들어 CYP2D6 유전자는 *1, *2, *4, *10 등 수십 가지의 allele을 가지며, 어떤 allele을 보유하느냐에 따라 효소의 활성도가 달라집니다.

🔬

유전자형 (Genotype)

개인이 실제로 보유한 유전자 구성을 의미합니다. 사람은 각 유전자를 부모로부터 하나씩 물려받아 두 개의 allele 쌍으로 이루어진 genotype을 가집니다.

⚗️

표현형 (Phenotype)

유전자형(Genotype)으로부터 예측되는 실제 기능적 특성을 의미합니다. 약물유전체학에서는 특정 유전자형이 해당 효소의 활성도를 어느 정도로 나타내는지를 phenotype으로 표현합니다.

// 04 — 05

표현형 변환 및 스코어링 시스템

Activity Score는 개인이 보유한 두 allele 각각의 기능적 활성도를 수치화하여 합산한 값입니다. 각 allele은 기능 수준에 따라 일정한 점수(0, 0.5, 1 등)가 부여되며, 두 allele의 점수를 더한 총합으로 해당 개인의 효소 활성도를 정량적으로 나타냅니다.

Activity Score (AS) = ValueAllele1 + ValueAllele2

계산된 AS 값에 따라 최종 Phenotype이 분류됩니다.

// 06

글로벌 표준 CPIC 가이드라인 연계

CPIC(임상약물유전체학 구현 컨소시엄) 가이드라인이란?

CPIC(Clinical Pharmacogenetics Implementation Consortium)은 전 세계의 유전체 연구 증거를 기반으로 '환자의 유전자 검사 결과가 나왔을 때, 의사가 처방을 어떻게 변경해야 하는가?'에 대한 구체적이고 동기화된 가이드라인을 제공하는 국제 표준 기구입니다.

🌐 공식 웹사이트: cpicpgx.org

CPIC이 제공하는 4가지 핵심 정보
📋

Allele 정의 및 기능적 분류 표준화

각 유전자의 allele별 기능 수준을 국제 기준에 따라 분류하고 표준화합니다.

🔄

유전자형 → 표현형 변환 기준 제공

Diplotype 기반의 phenotype 분류 알고리즘과 activity score 계산 기준을 제공합니다.

💊

임상 약물 처방 권고사항 발표

표현형에 따른 약물 용량 조정 및 대체 약물 처방 가이드를 제시합니다.

🌏

인종별 Allele Frequency 데이터

동아시아인을 포함한 인종별 대립유전자 빈도 데이터를 수집·제공합니다.

왜 CPIC 가이드라인으로 데이터베이스를 구성했는가?
✅

표준화된 분류 체계

CPIC은 각 유전자의 allele별 activity score, phenotype 분류 기준을 국제적으로 표준화하여 일관성과 신뢰성을 충족하는 데이터를 제공합니다.

📊

인종별 Allele Frequency

인종별 대립유전자 빈도 테이블을 함께 제공하므로 한국인을 포함한 동아시아인 특이적 위험인자 스크리닝이 가능합니다.

🔬

임상적 근거 기반

CPIC은 peer-reviewed 연구를 바탕으로 작성되며 지속적으로 업데이트됩니다. 검증된 데이터를 사용하여 프로젝트의 신뢰도를 높일 수 있었습니다.

🔓

공개 접근 가능한 데이터

CPIC의 모든 가이드라인과 데이터는 무료로 공개되어 있어, 학술 및 교육 목적의 프로젝트에 적합하게 활용할 수 있었습니다.

// 07

CYP.OR1 실제 구현 모델

저희 개발 팀은 CPIC guideline 및 데이터를 전처리하여 데이터베이스 구조를 설계했습니다.

01
CPIC Allele Definition Table 데이터 가공 및 적재

CPIC 공식 가이드라인에서 제공하는 allele 정의 테이블을 전처리하여 데이터베이스에 구조화된 형태로 적재합니다.

02
Activity Score 계산 모듈

환자 유전형 입력 시 알고리즘 기반 Activity Score 계산 모듈이 작동하여 두 allele의 기능 점수를 합산합니다.

03
Diplotype-to-Phenotype Mapping

Diplotype-to-Phenotype Mapping Tables를 거쳐 최종 대사 표현형(UM / RM / NM / IM / PM)이 확정됩니다.

04
인종별 빈도 시각화

CPIC Frequency Data와 결합하여 환자가 속한 인종(예: 동아시아인) 내 대사 취약성 통계를 화면에 시각화합니다.
